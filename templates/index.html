Features

  • Create: Add new sensor readings manually
  • Read: View device data and readings
  • Update: Edit existing readings
  • Delete: Remove unwanted readings
  • REST API: Programmatic access for IoT devices

API Endpoints

  • POST /api/data - Receive IoT data
  • GET /api/data/<device_id> - Get device readings
  • GET /api/health - Health check
  • GET /api - API information

Getting Started

Your IoT devices can send data to POST /api/data endpoint. The system automatically creates the database structure and manages all sensor data efficiently.

For MicroPython devices, check the micropython_client.py example in your project directory.
